#AverageOS
##Yet another 32-bit operating system made in C...

## Building:
Simply run ```make``` and it will build (Linux only).
> [!NOTE]
> Make sure you have:
> ```gcc``` and ```gcc-multilib```.
> ```grub```.
> ```nasm```.

## Testing:
Just make sure you have ```qemu-system-i386``` and run ```make run```.
