# AverageOS
Yet another 32-bit operating system made in C...
![alt text](logo.png)

## Building:
Simply run ```make``` and it will build (Linux only).
> [!NOTE]
> Make sure you have:
> ```gcc``` and ```gcc-multilib```,
>  ```grub```,
>  ```nasm```.

## Testing:
You can use ```VirtualBox``` or run ```make run``` to launch ```qemu-system-i386```.
